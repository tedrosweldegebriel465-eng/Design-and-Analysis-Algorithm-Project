# 📊 Visualization Output Directory

This directory contains all visualizations generated by the Ethiopian GPS Navigation System.

## Directory Structure

## File Naming Convention

Files are automatically named using the pattern:
`{type}_{timestamp}_{description}.{extension}`

Examples:
- `map_interactive_20240127_143022_ethiopia_network.html`
- `graph_spring_20240127_143022_shortest_path.png`
- `chart_degree_distribution_20240127_143022.png`

## Generated Visualizations

### 🗺️ Maps (`/maps`)
- `ethiopia_road_network.html` - Interactive map with all cities and roads
- `ethiopia_shortest_path_{source}_{dest}.html` - Map showing specific shortest path
- `ethiopia_heatmap_population.html` - Population density heatmap
- `ethiopia_road_conditions.html` - Map colored by road condition
- `ethiopia_regions.html` - Map highlighting Ethiopian regions

### 📈 Graphs (`/graphs`)
- `graph_spring_layout.png` - Network graph with spring layout
- `graph_geographic_layout.png` - Network graph with geographic coordinates
- `graph_path_comparison.png` - Multiple paths comparison
- `graph_degree_distribution.png` - Degree distribution histogram
- `graph_centrality.png` - Node centrality visualization

### 📊 Charts (`/charts`)
- `chart_distance_distribution.png` - Distribution of road distances
- `chart_cities_by_region.png` - Bar chart of cities per region
- `chart_algorithm_comparison.png` - Performance comparison chart
- `chart_scalability.png` - Algorithm scalability plot
- `chart_memory_usage.png` - Memory usage comparison

### 🎥 Animations (`/animations`)
- `animation_dijkstra_steps.gif` - Step-by-step algorithm animation
- `animation_path_expansion.gif` - Shortest path tree expansion

## Usage

These visualizations are automatically generated when you:
1. Run the main application
2. Execute the demonstration notebook
3. Use the visualization module directly

```python
from src.visualization.map_plotter import MapPlotter
from src.visualization.graph_viz import GraphVisualizer

# Save visualizations to this directory
plotter = MapPlotter(network)
plotter.create_interactive_map(output_file='visualizations/maps/my_map.html')

viz = GraphVisualizer(network)
viz.draw_graph(save_path='visualizations/graphs/my_graph.png')
